# Requirement
Node v7.6.0 or greater

# Setting up
## Install Puppeteer:
~~~sh
npm i puppeteer
~~~

## Initialize NPM:
~~~sh
npm init
~~~

# Running
~~~sh
cd .\ScreenShot
~~~
    
~~~sh
mkdir Sample
~~~
    
~~~sh
node getScreenShots.js
~~~
